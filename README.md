# 设计系统规范仓库

## 这是什么？
这里是产品团队的设计系统规范，包含颜色、组件、交互规则等所有设计相关信息。

## 文件说明
- `design-tokens.json`：机器可读的设计令牌，包含所有色值、尺寸等具体数据
- `AGENTS.md`：AI 可读的设计指南，包含使用原则和组件规范

## 如何使用
### 产品经理使用流程
1. 在 VSCode 中编写 PRD 或需求描述
2. 调用 AI 助手（如 Claude Sonnet）
3. AI 会自动读取本仓库的规范文件
4. AI 生成符合规范的 Vue 组件代码

### 规范版本更新
当设计师更新规范后，本仓库会更新版本号。请在项目中指定使用的版本：
- 最新版：始终跟随 main 分支
- 固定版本：使用 Git tag，如 v1.0.0
